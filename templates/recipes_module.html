This is really similar to the What can I make section. You can filter down the boxes, and combine your filters with logical operators, courtesy of Logipar. Unfortunately, they read awkwardly here.

I'm not sure about how I want this section to work in general. It may not be overly useful as more than a toy. We'll see. If you have any requests, let me know.

Like the What can I make tab, you can do:

  • Foodtype is veggie to see all the vegetarian options. The other values seem to be "meat", "goodies", and empty. You use this option with any of the fields, which are:
    • Foodtype
    • Sanity
    • Hunger
    • Health
    • Perishtime
    • Cooktime
    • Temperature
    • Temperatureduration
    • Priority – if your ingredients match more than one test function, presumably the one with the highest priority will be selected.
    • Name
    • TestM – the test function to see if this recipe can be generated from your ingredients.
  • Less than [amount] [field], [amount field], Greater than [amount field] can be used with any numeric field: Sanity, Hunger, Health, Perishtime, Cooktime, Temperature, Temperatureduration, Priority. The same aliases also apply (fewer than and more than).
  • Name contains meat will show all recipes that have "meat" in their name.
